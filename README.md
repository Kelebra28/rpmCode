## Installation

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Clone the project

   ```sh
   git clone "https://github.com/Kelebra28/rpmCode.git"
   ```

3. Once clone it, you can access to edit or run in local.

   ```sh
   cd rpm-landing
   gatsby develop
   ```
